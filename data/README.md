# 数据目录

这个目录包含项目的示例数据和数据说明。

## 文件说明

- `sample_data.json` - 示例TOC格式数据，用于测试和演示
- 其他数据文件根据需要添加

## 数据格式

输入数据应为TOC JSON格式，每个文档包含以下字段：

```json
{
  "segment_id": "唯一标识符",
  "title": "段落标题",
  "level": 1,
  "order": 1,
  "text": "段落文本内容",
  "state": "州标识",
  "language": "语言标识"
}
```

## 使用方法

将您的数据文件放在此目录中，然后在主程序中指定路径即可。
