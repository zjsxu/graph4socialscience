#### 2. 配置管理 ⚙️
- **2.1 设置处理参数**: 配置语言、频率阈值等
- **2.2 查看当前配置**: 显示所有配置选项

#### 3. 处理功能 🔧
- **3.1 完整管线处理**: 运行完整的分析流程
- **3.2 文本处理**: 单独执行文本预处理
- **3.3 词组抽取**: 单独执行词组抽取
- **3.4 停词发现**: 单独执行停词发现
- **3.5 图构建**: 单独执行图构建
- **3.6 网络分析**: 单独执行网络分析

#### 4. 结果管理 📊
- **4.1 生成报告**: 创建分析报告
- **4.2 查看结果文件**: 浏览输出文件
- **4.3 导出数据**: 导出处理结果

#### 5. 帮助和工具 🛠️
- **5.1 创建示例数据**: 生成测试用的示例文件
- **5.2 系统信息**: 查看系统和管线状态
- **5.3 帮助文档**: 显示使用帮助

## 🎯 支持的文件格式

### 输入文件
- **JSON文件**: 结构化文档数据
  ```json
  {
    "segment_id": "doc_001",
    "title": "文档标题",
    "text": "文档内容...",
    "state": "CA",
    "language": "english"
  }
  ```
- **文本文件**: 纯文本内容（自动转换为JSON格式）

### 输出文件
- **JSON**: 结构化数据
- **CSV**: 表格数据
- **PNG**: 可视化图表
- **Markdown**: 分析报告

## ⚙️ 配置选项

- **语言设置**: auto（自动检测）、english、chinese
- **最小词组频率**: 控制词组筛选阈值
- **动态停词发现**: 启用/禁用自动停词识别
- **生成可视化**: 启用/禁用图表生成
